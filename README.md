# このアプリケーションについて

フィンガースナップ、いわゆる指パッチンを検出します。
指パッチンを何かのトリガーとすることで、非モテ男子をモテ男子に変えることができます。
実際に私が開発した、指パッチンをトリガーとしたものたちを紹介します。
